Доступные end-points:


.../articles - получение всех записей, находящихся в базе данных

.../articles/{article_id} - получения записи из базы данных с id равным article_id

.../new_article - для добавления записи в базу данных

.../update_article/{article_id} - обновление записи в базе данных с id равным article_id

.../delete_article/{article_id} - удаление записи в базе данных с id равным article_id
